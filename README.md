# road-damage-thesis-exp

Private experiment registry for the road-damage thesis experiments.

This repository stores reusable scripts, configuration files, operation records, path indexes, dataset/statistical summaries, screening metadata, and selected final-result references.

Large image datasets, full labels directories, generated candidates, training runs, caches, credentials, and non-final model weights are intentionally excluded.
